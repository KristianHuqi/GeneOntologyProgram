TOOLS MENU

  • Visualise general statistics

    • Here you can visualise: graph data, number of genes, number of annotations, top 10 more annotated genes, top 10 more populated GO terms.

  • Search by GO ID

    • Here you can visualise: definition, synonyms, if it is obsolete (or not) and who replaced it, ancestors and descendants.

  • Search by gene

    • Here you can visualise: annotations; for each annotation GO ID, GO name, namespace, evidence code and aspect. You will see a short overview of the annotations of that specific gene, then you can take the GO: ID and use the tool 'Search by GO ID'.

  • Similarity

    • Cosine similarity: using genes it calculates the exact proportion of shared biological functions. It divides the number of shared GO annotations by the total distinct annotations of both genes. Instead using GO terms measures the exact overlap of ancestral lineages. It divides the shared ancestors by the total unique ancestors of both terms up to the root.

    Jaccard similarity: using genes it evaluates the functional alignment (trajectory) between genes. It is less penalized by differences in annotation volume (magnitude), focusing on whether the genes operate within the same biological macro-area, instead using GO terms evaluates the topological trajectory within the ontology. It identifies if terms derive from the same primary lineage, largely ignoring differences in their absolute hierarchical depth.

  • Neighbourhood analysis

    • Explore the ontology graph: enter a GO term ID and a depth level to retrieve its surrounding network of related terms.
